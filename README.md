# sinocolonial

## About

Inspired by H.W. Knight & Son catalogue of metal numerals published in 1927. The catalogue is in the Public Domain. 

The [catalogue page at archive.org](https://archive.org/details/CatalogueAndPriceListNo.27AluminumWhiteMetalBrassPatternLetters/page/n61/mode/2up) shows the numerals on page 60.

This project is currently a work-in-progress (WIP).
